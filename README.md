# 🏡 HavenStay – Full-Stack Rental Booking Platform

HavenStay is a full-stack rental booking platform that enables users to discover, list, review, and manage rental properties through a seamless and user-friendly web experience. Inspired by modern accommodation marketplaces, the platform provides secure authentication, property management tools, review systems, and role-based access control to create a reliable and scalable booking ecosystem.

Built using Node.js, Express.js, MongoDB, and EJS, the application follows industry-standard development practices including RESTful APIs, MVC architecture, server-side validation, and secure user authentication.

## 🚀 Features

### 🔐 Secure Authentication & Authorization

* User registration and login using Passport.js
* Secure session-based authentication
* Role-based access control for managing platform resources
* Protected routes for authorized users only

### 🏠 Property Listings Management

* Create, edit, update, and delete rental property listings
* Upload and manage listing images through Cloudinary
* Detailed property information with descriptions and pricing
* Responsive property browsing experience

### ⭐ Reviews & User Feedback

* Add, edit, and delete property reviews
* Associate reviews with authenticated users
* Improve listing credibility through user-generated feedback

### 📸 Cloud-Based Image Management

* Cloudinary integration for image storage and optimization
* Efficient handling of property images
* Reliable cloud-hosted media delivery

### ⚡ Backend Architecture

* RESTful API-driven workflows
* MVC (Model-View-Controller) design pattern
* Modular and maintainable backend structure
* Server-side validation for improved security and data integrity

### 📱 Responsive User Experience

* Mobile-friendly EJS templates
* Clean and intuitive user interface
* Consistent experience across devices and screen sizes

## 🏗️ Project Architecture

The application follows the MVC (Model-View-Controller) pattern to maintain a clear separation of concerns:

* **Models** manage database schemas and business data.
* **Views** render dynamic EJS templates for users.
* **Controllers** handle application logic and request processing.
* **Routes** define API endpoints and user workflows.

This architecture improves code organization, scalability, and long-term maintainability.
